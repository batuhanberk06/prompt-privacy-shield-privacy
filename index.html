Privacy Policy

Effective date: March 19, 2026

Prompt Privacy Shield is a browser extension designed to help users detect and mask sensitive information before sending prompts to AI tools. This privacy policy explains what the extension does, how it processes data, and what rights and protections users have.

What the extension does

The extension scans text provided by the user in order to identify common sensitive data patterns such as email addresses, phone numbers, IBANs, payment-related data, API keys, card numbers, crypto wallet addresses, SSN and ABA routing numbers, VAT numbers, and custom protected words defined by the user.

On supported AI websites (including ChatGPT, Claude, Google Gemini, DeepSeek, Mistral, and Grok), the extension displays a local "Clean Prompt" button near the prompt input area. When the user clicks this button, the extension sanitizes the prompt directly in the page before it is sent. The extension also includes smart detection that may work on AI sites not explicitly listed.

How data is processed

All processing is performed locally in the user's browser using a shared detection engine. Prompt Privacy Shield does not send scanned text or prompts to any external server. No AI models or cloud services are used for text analysis.

Data the extension may process

  • User-provided text pasted into the extension popup
  • User-selected text from a web page when the user uses the extension's right-click "Sanitize & Copy" action
  • Prompt text that the user chooses to sanitize using the Clean Prompt button on supported AI websites
  • User preferences such as masking style, selected region, enabled detection options, and custom protected words

Storage

The extension stores user settings locally in the browser using extension storage. These settings may include masking preferences, region settings, enabled scan categories, and custom protected words defined by the user. No user content or sanitized text is stored.

What we do not do

  • We do not sell user data
  • We do not transfer scanned content or prompts to remote servers for processing
  • We do not use user data for advertising
  • We do not use user data for creditworthiness or lending decisions
  • We do not collect or store the text that users sanitize
  • We do not use remote code execution or dynamically loaded scripts
  • We do not track user behavior or browsing activity

Permissions

Prompt Privacy Shield requests only the permissions required for its core functionality:

  • activeTab — to work with text from the active tab when the user interacts with the extension
  • storage — to save user preferences and custom protected words locally
  • scripting — to run local sanitization logic in the page context for the Clean Prompt button and context menu
  • contextMenus — to provide the "Sanitize & Copy" right-click action
  • Host access on supported AI sites — to display the Clean Prompt button and sanitize prompts locally in place

Third-party services

Prompt Privacy Shield does not use remote code or third-party services to process user text. If the extension includes optional links to third-party pages such as feedback forms, support pages, donation services, or external websites, those services operate under their own privacy policies.
