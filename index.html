Prompt Privacy Shield

Privacy Policy

Effective date: March 15, 2026

Prompt Privacy Shield is a browser extension designed to help users detect and mask sensitive information before sending prompts to AI tools.

What the extension does

The extension scans text provided by the user or text explicitly selected by the user in order to identify common sensitive data patterns such as email addresses, phone numbers, IBANs, payment-related data, API keys, card numbers, crypto wallet addresses, and custom protected words.

How data is processed

All processing is performed locally in the user’s browser. Prompt Privacy Shield does not send scanned text to our servers.

Data the extension may process

  • User-provided text pasted into the extension popup
  • User-selected text from a web page when the user uses the extension’s right-click action
  • User preferences such as masking style, selected region, enabled detection options, and custom protected words

Storage

The extension stores user settings locally in the browser using extension storage. These settings may include masking preferences, region settings, enabled scan categories, and custom protected words defined by the user.

What we do not do

  • We do not sell user data
  • We do not transfer scanned content to remote servers for processing
  • We do not use user data for advertising
  • We do not use user data for creditworthiness or lending decisions

Permissions

Prompt Privacy Shield requests only the permissions required for its core functionality, including access to the active tab when the user interacts with the extension, local storage for saving settings, scripting for on-demand sanitization actions, and context menus for the “Sanitize & Copy” feature.

Third-party services

Prompt Privacy Shield does not use remote code to process user text. If donation, support, or feedback links are added in the future, those services may have their own privacy policies.

Changes to this policy

This privacy policy may be updated from time to time to reflect product changes, compliance needs, or clarification improvements. The latest version will be published on this page.
